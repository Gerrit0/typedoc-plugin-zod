typedoc-plugin-zod

Improves display of types created with Zod's z.infer and z.input types.

Usage

npm install --save-dev typedoc-plugin-zod

// typedoc.json
{
    "plugin": ["typedoc-plugin-zod"]
}

See an example of this plugin in action.
